# Scheduler

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A scheduling application built with Next.js, TypeScript, and Gemini AI integration. Originally created using Google AI Studio.

**Live Demo:** [scheduler-neon.vercel.app](https://scheduler-neon.vercel.app)

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database:** Prisma ORM
- **AI:** Google Gemini API
- **Hosting:** Vercel

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key

3. Run the app:
   ```bash
   npm run dev
   ```
